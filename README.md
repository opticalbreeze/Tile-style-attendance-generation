# タイルスタイル勤務スケジュール管理システム

FlaskベースのWebアプリケーションで、ドラッグ&ドロップで勤務スケジュールを管理できます。

## 機能

- 給料計算月度（前月16日～当月15日）のスケジュール表示
- ドラッグ&ドロップで勤務種別を配置
- 勤務種別：24A、24B、日勤、夜勤、明、休、有休
- 24A/24B/夜勤の翌日に自動的に「明」を配置
- 右クリックでシフト削除
- 各日付の勤務種別ごとの人数集計
- スタッフ名カラムの固定表示
- 土日の色分け表示

## セットアップ

```bash
# 依存関係のインストール
pip install -r requirements.txt

# アプリケーションの起動
python app.py
```

ブラウザで `http://localhost:5000` にアクセスしてください。

## 使用方法

1. 右側の勤務種別タイルをドラッグして、該当する日付セルにドロップ
2. 配置済みのシフトを右クリックして削除
3. 表の下部に各日付の勤務種別ごとの人数が自動集計されます

## 技術スタック

- Python 3.x
- Flask
- HTML/CSS/JavaScript
